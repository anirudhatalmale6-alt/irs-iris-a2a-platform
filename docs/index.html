Generated JWT
📋 A2A OAuth Flow (Pub 5718, Section 3)
  1. Generate Client JWT (signed with your private key, identifies your application)
  2. Generate User JWT (signed with your private key, identifies the user/taxpayer)
  3. Send both JWTs to IRS Authorization Server to request an access token
  4. Use the access token to call IRS IRIS API endpoints
  5. Access token expires - repeat steps 1-3 when needed
